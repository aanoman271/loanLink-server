### Server-Side (Node/Express)

- express
- mongoose
- cors
- jsonwebtoken
- dotenv
- stripe
- cookie-parser
- (Additional: multer for image uploads if implemented)

## Installation & Setup

1. Clone client and server repositories.
2. Install dependencies: `npm install` in both folders.
3. Set up `.env` files with required keys (Firebase config, Mongo URI, JWT secret, Stripe keys).
4. Run locally: `npm start` (client) and `node index.js` (server).
